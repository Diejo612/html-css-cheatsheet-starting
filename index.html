Class and ID

We could select element by Class or ID, for instance:

<!-- element with Class atribute -->

<p class='paragraph' >
Lorem ipsum dolor sit amet consectetur, adipisicing elit.
</p>

<!-- element with ID atribute -->

<p id='example' >
Lorem ipsum dolor sit amet consectetur, adipisicing elit.
</p>

/* Class selector */

.paragraph {
color: blue;
font-size: 11px;
}

/* ID selector */

#example {
color: green;
font-size: 11px;
}

CSS Type selector

We can also select by HTML element type:

<!-- H1 element -->

<H1> Welcome to my blog </H1>

/* Selecting elements of type H1*/

h1 {
text-align: center;
font-weigth: bold;
}

Multipe elements selector

We can select multiple elements by groups:

<!-- H1 element -->

<H1> Welcome to my blog </H1>

<!-- element with Class atribute -->

<p class='paragraph' >
Lorem ipsum dolor sit amet consectetur, adipisicing elit.
</p>

<!-- element with ID atribute -->

<p id='example' >
Lorem ipsum dolor sit amet consectetur, adipisicing elit.
</p>

/* Selecting multiple element with a common rule */

h1, .paragraph, #example {
font-family: Verdana;
background-color: skyblue;
}
